# firefly-settings

The system settings app for [Firefly Zero](https://fireflyzero.com/). Written in Rust using [the official Rust SDK](https://github.com/firefly-zero/firefly-rust).

## Installation

1. [Install firefly-cli](https://github.com/firefly-zero/firefly-cli?tab=readme-ov-file#installation)
1. Run `firefly_cli import sys.settings`
